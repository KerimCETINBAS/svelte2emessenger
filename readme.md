## Svelte Socket.io e2e encrypted web messenger tutorial
___

clone this repo

```git
git clone https://github.com/kerimcetinbas/svelte2emessenger.git svelte2e
```

### build client app

go client folder

```shell
cd ./svelte2e/client

```

install dependencies

```node
// install with npm
npm install
// or with yarn 
yarn install
```

build app
```node
// build with npm
npm run build

// or with yarn
yarn build
```

## prepare server

```shell
cd ../server
```

install dependencies

```node
// install with npm
npm install
// or with yarn 
yarn install
```

run server deamon

```node
node server.js
```


now you can visit http://localhost:3000 and play with the app
